# Interactive Digital Archive of Archaeological Heritage of Kyrgyzstan

This repository contains a prototype of an interactive digital archive
with electronic visualization of the archaeological heritage of Kyrgyzstan.
The project is developed for scientific and educational purposes within
the framework of a Master’s thesis in History (Digital Archaeology / Digital Humanities).

## Project Objectives
- To structure archaeological data in a digital format;
- To visualize archaeological sites using interactive web maps;
- To demonstrate the potential of digital technologies for archaeological research and education.

## Technologies Used
- HTML, CSS, JavaScript
- Leaflet.js for interactive maps
- Python (GeoPandas) for spatial analysis
- JSON / CSV as a structured digital database format

## Repository Structure
- `data/` – structured archaeological data
- `frontend/` – web interface and visualization
- `analysis/` – spatial analysis and data processing
- `docs/` – project documentation

## Academic Context
This project represents a conceptual and technical model of a digital archaeological archive.
It does not aim to replace official state databases but demonstrates methodological
approaches to digital preservation and visualization of archaeological heritage.
